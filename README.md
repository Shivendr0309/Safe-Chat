# Safe Chat

Safe Chat is a full-stack web application designed to detect and prevent cyberbullying in real-time chats.

## Tech Stack
Frontend: React.js  
Backend: Node.js + Express  
Database: MongoDB  

## Features
- Real-time chat monitoring
- AI-based toxic message detection
- User authentication

## Setup
1. Clone the repository
2. Install dependencies
3. Add environment variables
4. Run backend and frontend
